## What This Is

A single-file, zero-dependency landing page for Social Vows — a wedding content creation service specializing in same-day and next-day Reels delivery. Built for conversion: hero → pain point → flip → packages → logistics → contact.

**Service:** Social Vows captures and edits wedding content, delivering edited videos and Reels within 24 hours of the event.

**Markets:** Houston · Dallas · Austin · California

---

## Tech

- Pure HTML/CSS/JS — no build step, no framework
- `clamp()`-based fluid typography system for all screen sizes
- CSS custom properties for theming (warm cream/terracotta editorial palette)
- `Instrument Serif` + `Manrope` font pairing via Google Fonts
- Deployed as a static site
